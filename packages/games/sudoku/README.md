# 多阶数独游戏 (Multi-tier Sudoku)

这是一个基于 React + TypeScript + Vite + Tailwind CSS 开发的数独游戏，支持多种阶数与智能辅助功能。

## 功能特性

- **多阶支持**：目前支持 4阶 (2x2)、6阶 (2x3)、8阶 (2x4) 和 9阶 (3x3) 数独。
- **可选难度**：提供简单、中等、困难三种挖空比例，满足不同水平玩家需求。
- **智能高亮**：
    - **十字高亮**：自动选中当前单元格所在的行、列、宫。
    - **同数高亮**：点击数字时，全盘相同的数字都会高亮显示，方便观察。
    - **冲突高亮**：实时检测并高亮显示行、列、宫内的重复数字。
- **撤销与重做**：支持完整步数回溯，包括填数和标记操作。
- **存疑标记**：支持右键标记“存疑”数字，辅助逻辑推理。
- **响应式 UI**：适配不同屏幕尺寸，提供直观的数字面板输入与帮助提示。

## 技术栈

- **框架**: [React 19](https://react.dev/)
- **构建工具**: [Vite 8](https://vite.dev/)
- **样式**: [Tailwind CSS 4](https://tailwindcss.com/)
- **语言**: [TypeScript](https://www.typescriptlang.org/)
- **包管理**: [pnpm](https://pnpm.io/)

## 如何运行

1. 安装依赖:
   ```bash
   pnpm install
   ```

2. 启动开发服务器:
   ```bash
   pnpm dev
   ```

3. 构建生产版本:
   ```bash
   pnpm build
   ```

## 操作指南

- **填数**：左键选中格子，点击下方数字面板。
- **存疑标记**：在已填入数字的格子上点击 **右键**，可将其标记为存疑（橙色斜体），再次右键解除。
- **清除**：点击“清除”按钮或填入新数字会同步移除该格子的存疑标记。
- **撤销/重做**：点击对应按钮可回溯操作历史。
- **帮助**：将鼠标悬停在标题旁的 **?** 图标上可查看即时帮助。

## 游戏规则 (6阶示例)

- 每一行、每一列以及每一个 **2x3** 的宫格内，必须包含数字 1-6 且不能重复。
- 灰底数字为系统生成（锁定），不可修改。
- 蓝底数字为正式填入，橙色斜体为存疑标记。
- 红色高亮背景表示存在冲突。
